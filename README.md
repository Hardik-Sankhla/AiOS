# AIOS - Autonomous AI Operating System

Lightweight AI automation framework for Android Termux.

## Quick Start

./scripts/bootstrap.sh

## Architecture

- **agents/** - Hermes agent orchestration
- **services/** - Core service layers
- **memory/** - Long-term memory and state
- **scripts/** - Automation and git workflows
- **configs/** - Configuration and settings
- **docs/** - Documentation

## Requirements

- Python 3.13+
- Android Termux
- SSH access to GitHub
